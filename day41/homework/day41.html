Front-end დეველოპმენტი უკავშირდება ვებსაიტების და აპლიკაციების ვიზუალურ მხარეს, ანუ იმას, რასაც მომხმარებელი ხედავს და interacts.

What is Frontend?

Front-end არის ვებსაიტების დეველოპმენტის პროცესი, რომელიც ასოცირდება ყველა იმ ელემენტთან, რასაც მომხმარებელი ხედავს და interacts. ეს მოიცავს HTML-ის, CSS-ის და JavaScript-ის გამოყენებას.

HTML არის სტრუქტურული ენა, რომელიც განსაზღვრავს გვერდის სტრუქტურას და კonten ტექსს. CSS გამოიყენება ვებ-გვერდის სტილის შესაქმნელად, ხოლო JavaScript დამატებით ფუნქციონალობას სძენს, მაგალითად, გვერდის დინამიკურად დატვირთვას.

HTML - რა არის HTML?

HTML (HyperText Markup Language) არის ვებ-გვერდების სტრუქტურის შექმნისთვის გამოყენებული მარკაპის ენა. HTML სტრუქტურას ქმნის სხვადასხვა ელემენტებისგან, როგორიცაა პარაგრაფები, სურათები, გრაფიკები და სხვა.

HTML-ის ისტორია

HTML პირველად შეიქმნა 1991 წელს ტიმ ბერნერს-ლიცის მიერ. იგი გახდა ვებსაიტების შექმნის უმნიშვნელოვანესი ნაწილი, რომელიც მუდმივად ვითარდებოდა და დღესდღეობით კიდევ უფრო ძლიერ შესაძლებლობებს გვთავაზობს.
